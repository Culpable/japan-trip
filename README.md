# Japan Trip 2026

Interactive itinerary for Hong Kong, Shenzhen and Japan from 1–15 August 2026.

The site is published through GitHub Pages from `index.html` on the `main` branch.

The static site includes a 45-day browser-local access gate, automatic trip-day state, hotel and booking utilities, per-day maps and weather links, and offline PWA caching. The access gate prevents casual viewing but is not a substitute for server-side authentication because GitHub Pages serves public static source files.
